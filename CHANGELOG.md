## [0.2.0] - 2024-11-06

### Added
- Digital twin visualization with isometric 2D floor plan views
- Image transformation system using Gemini Imagen 3
- Design studio UI (React + Vite)
- Multi-floor plan detection and support
- Room-to-image linking system
- Google Cloud Storage integration
- Comprehensive API endpoints for design transformations
- Project structure documentation
- Improved README with badges and table of contents

### Changed
- Reorganized project structure for better maintainability
- Updated documentation organization in `docs/` directory
- Improved database schema for multi-floor support
- Enhanced floor plan analysis with better room detection

### Removed
- Cleaned up 50+ redundant markdown files from root directory
- Removed old test files from root (moved to `tests/`)
- Removed outdated utility scripts
- Cleaned up generated images and old exports

- Floor plan analysis capabilities
- Room image tagging system
- SQLite and PostgreSQL support
- Alembic database migrations
- Basic frontend structure (Next.js)
- Docker configuration for development and production
- Comprehensive documentation structure
- Jupyter notebooks for analysis and experimentation

### Infrastructure
- FastAPI application setup
- Database configuration (SQLite/PostgreSQL)
